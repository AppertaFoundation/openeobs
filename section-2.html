Menu

Styling for the main menu

Modifiers

.header

Defines the navigation as floating at the top of the page

.header-main

Contains utility items such as logo, logout etc

.header-main .logo

Logo (floated left)

.header-main .header-meta

Contains utility buttons (floated right)

.header-menu

Contains the main navigation items used in conjuction with inline button style

.header-menu li .urgent-badge

Used to show notifications for stand-in

#obsForm .obsSelectField .input-body
